RxJava学习手记

标签（空格分隔）： android RxJava

0x00

看到github上有很多项目都是使用了RxJava的，不太懂，感觉还是姿势不够，需要学习一个～，在网上找了一些别人的Blog学习了一下，然后自己动手写代码消化一下～在这里主要感谢@扔物线的博文和@_SOLID的博文。

下面直接叙述一下使用RxJava的过程吧，看上去会相对“顺滑”。

0x01

RxJava的核心开发者的github主页里可以找到RxJava和RxAndroid的项目。android开发者在使用的时候，可以引入依赖：

compile ‘io.reactivex:rxandroid:1.2.1’
compile ‘io.reactivex:rxjava:1.1.6’
//引用时间：20160617，最新版本

RxJava页面对项目的介绍：

RxJava is a Java VM implementation of Reactive Extensions: a library for composing asynchronous and event-based programs by using observable sequences.’

It extends the observer pattern to support sequences of data/events and adds operators that allow you to compose sequences together declaratively while abstracting away concerns about things like low-level threading, synchronization, thread-safety and concurrent data structures.

  • Zero Dependencies
  • < 1MB Jar
  • Java 6+ & Android 2.3+
  • Java 8 lambda support
  • Non-opinionated about source of concurrency (threads, pools, event loops, fibers, actors, etc)
  • Async or synchronous execution
  • Virtual time and schedulers for parameterized concurrency

可以看出它是一个基于event的、使用观察者模式的一个异步库。使用RxJava进行异步操作的时候代码非常简单易懂
